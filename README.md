# ANPR-system
Automobile NUmber Plate Recognition System

models folder contains best.onnx, lprnet_uzbek_pro.onnx and lprnet_uzbek_pro.onnx.data files,
best.onnx -> DETECTION: YOLO11 fine tuned model with 20k car images, and converted into onnx,
lprnet_uzbek_pro.onnx -> RECOGNITION: lprnet model fine tuned with uzbek number plates.
